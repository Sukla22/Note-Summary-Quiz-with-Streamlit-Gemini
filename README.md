# Note Summary & Quiz Generator with Gemini

[![Streamlit](https://img.shields.io/badge/Streamlit-FF4B4B?style=for-the-badge&logo=streamlit&logoColor=white)](https://streamlit.io/)
[![Gemini](https://img.shields.io/badge/Google_Gemini-4285F4?style=for-the-badge&logo=google&logoColor=white)](https://ai.google.dev/)

Transform your notes into concise summaries and interactive quizzes using Google Gemini AI. Perfect for students, researchers, and professionals who want to master their study materials efficiently.

## 🚀 Live Demo

Try it live: [https://note-summary-quiz-generator-with-gemini.streamlit.app/](https://note-summary-quiz-generator-with-gemini.streamlit.app/) [web:1]

## ✨ Features

- **📝 Smart Note Summarization** - Get concise, accurate summaries of your notes
- **❓ Auto Quiz Generation** - Multiple choice, true/false, and short answer questions
- **⚡ Real-time Processing** - Powered by Google Gemini AI
- **📱 Mobile-friendly Interface** - Works perfectly on all devices
- **🎯 Customizable Output** - Adjust summary length and quiz difficulty

## 🛠️ Quick Start (Local Development)

### Prerequisites
- Python 3.8+
- Google Gemini API key

### Installation
```bash
git clone <your-repo-url>
cd note-summary-quiz-generator-with-gemini
pip install -r requirements.txt
```

### Setup
1. Get your [Gemini API key](https://aistudio.google.com/app/apikey)
2. Create `.streamlit/secrets.toml`:
```toml
GEMINI_API_KEY = "your-api-key-here"
```

### Run Locally
```bash
streamlit run app.py
```

## 📖 Usage

1. **Upload your notes** (PDF, TXT, or paste directly)
2. **Generate Summary** - Click "Summarize Notes"
3. **Create Quiz** - Select quiz type and difficulty
4. **Practice & Learn** - Test yourself with instant feedback
